# Documentation Set

This folder complements the preserved notebook and the existing `Methodology and Results.docx`. Each Markdown file is written so it can be incorporated into the Word report while also functioning as standalone project documentation.

## Files

- `01_theoretical_background.md`: conceptual rationale for entropy weighting, composite indicators, and clustering.
- `02_methodology.md`: reproducible description of the implemented pipeline and equations.
- `03_data_and_indicator_design.md`: interpretation of the workbook structure, dimensions, and indicator orientation.
- `04_workflow_and_reproducibility.md`: practical guide to running the project and understanding the file flow.
- `05_results_and_interpretation.md`: updated findings based on the current local run.
- `06_docx_revision_guide.md`: concrete recommendations for improving `Methodology and Results.docx`.

## How To Use These Files

1. Keep the notebook and the Word document in the repository.
2. Use the Markdown files as the report-ready source of truth for methodology and interpretation.
3. Copy or adapt the relevant sections into `Methodology and Results.docx` when preparing the final manuscript.
